# PersonalRaspiBot
A raspberry Pi robot which can be controlled using keyboard input.
w --> Up
a --> Left
s --> Down
d --> right

Parts required
  1. Raspberry Pi 3
  2. L298n motor driver
  3. 4 DC Motors with Wheels
  5. M2M, F2F, and M2F jumper wires
  6. Robot Frame to serve as exterior
